# Project Core
An interactive single-page course: "Zero to Live: A Guide to Building with Claude Code."

# Tech Stack
- Single HTML file (index.html)
- Tailwind CSS via CDN (no build step)
- Vanilla JavaScript (inline, no external files)
- Google Fonts: Inter, JetBrains Mono, Playfair Display

# Design System
- Background: deep navy (#0F1A2E), cards (#162032), title bars (#1E2D4A)
- Accent: terracotta (#E07A5F) for buttons, links, progress bar
- Highlight: gold (#F2CC8F) for checkmarks, callout borders, $ prompts
- Headings: Playfair Display (font-display class)
- Body: Inter | Code: JetBrains Mono

# Interactive Features
- Collapsible sections (Step 0 and Launch open by default)
- Scroll progress bar (fixed, top of page)
- Checklist with localStorage persistence
- Terminal/browser/editor CSS mockups with traffic-light dots
